# Oppenhomie (Bomberman)

A local 2-player Bomberman clone built with [Phaser 3](https://phaser.io/), TypeScript, and Vite.

> **Note:** Power-up spawn weights are set to 1.0 for testing. Adjust in `src/config/constants.ts` before release.

## Running the game

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

---

## Game flow

**Boot → Title (mode select) → Game**

- Title screen: click **2P Battle** or press **Enter** to start
- In-game: press **R** to restart the round
- Click the **⌂** home icon (top-left) anytime to return to the title screen

## Controls

| Action             | Player 1    | Player 2  |
|--------------------|-------------|-----------|
| Move               | WASD        | IJKL      |
| Normal bomb        | Left Shift  | Space     |
| Pierce bomb        | Q           | O         |
| Remote bomb (place/detonate) | E | U         |
| Restart            | R           |           |

## Power-ups

| Icon | Color | Effect |
|------|-------|--------|
| +1 Bomb | Yellow | +1 max bomb capacity (permanent) |
| +1 Range | Purple | +0.5 explosion range per pickup (permanent) |
| Speed Up | Green | Permanent speed boost (stacks with each pickup) |
| Bomb Kick | Orange | Walk into bombs to kick them sliding (permanent) |
| Full Fire | Red | Max explosion range instantly (permanent, very rare) |
| Pierce Bomb | Cyan | Gives 3 pierce bombs — blast goes through soft walls |
| Remote Bomb | Purple | Gives 3 remote bombs — no timer, detonate manually |
| Skull | Gray | 30-second curse: slow, reverse controls, or 1-tile range |

## Bomb types

| Type | Sprite | Behavior |
|------|--------|----------|
| Normal | Blue/Red (per player) | 2.5s fuse, standard explosion |
| Pierce | Cyan | Blast continues through soft walls |
| Remote | Dark purple | No auto-fuse, slow pulse, press key again to detonate all |
| Cursed | Any (placed while cursed) | Range locked to 1, blocks incoming explosion rays, transfers curse on hit instead of killing |

## Project structure

```
src/
├── main.ts                   # Phaser config, registers scenes
├── config/constants.ts       # Grid size, tile types, power-up types, tuning values
├── scenes/
│   ├── BootScene.ts          # Loads all assets, then starts TitleScene
│   ├── TitleScene.ts         # Mode select screen
│   └── GameScene.ts          # Game loop, grid, bombs, deaths, power-ups
└── entities/
    ├── Player.ts             # Movement, bomb placement, power-up collection, curse
    ├── Bomb.ts               # Countdown, explosion, chain reactions, sliding, bomb types
    └── PowerUp.ts            # Spawned on soft-wall destruction
```
